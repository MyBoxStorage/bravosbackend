# BRAVOS BRASIL - Backend API

Backend Node.js/Express para o e-commerce BRAVOS BRASIL.

## Tecnologias

- Node.js 20
- Express
- TypeScript
- Prisma
- PostgreSQL (Supabase)

## Deploy

Deploy automático via GitHub Actions para Fly.io.

## Variáveis de Ambiente

- `DATABASE_URL` - URL do PostgreSQL (Supabase)
- `MP_ACCESS_TOKEN` - Token do Mercado Pago
- `FRONTEND_URL` - URL do frontend
- `BACKEND_URL` - URL deste backend
- `ADMIN_TOKEN` - Token para rotas admin
